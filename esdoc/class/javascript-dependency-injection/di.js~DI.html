    public getInstance(contract: String, params: ...*): Object source

    Returns an instance for the given contract. Use <tt>params</tt> attribute to overwrite the default parameters for this contract. If <tt>params</tt> is defined, the singleton will be (re)created and its parameters are updated.

    Params:

    NameTypeAttributeDescription
    contract String

    name
    params ...*
    • optional

    constructor parameters which, if defined, replaces its default arguments (see {{#crossLink "DI/register:method"}}{{/crossLink}} )

    Return:

    Object

    Class instance

    Example:

    App.di.register("ajax", ["rest"]) ;
var ajax = App.di.getInstance("ajax") ;
ajax = App.di.getInstance("ajax", "rest", true) ;

    public register(contract: String, classRef: Class, params: Array, options: Object): Object source

    Register a class by creating a contract. Use {{#crossLink "DI/getInstance:method"}}{{/crossLink}} to obtain an instance from this contract. The <tt>params</tt> parameter is a list of contracts, and, if needed, normal constructor parameters can be mixed in.

    Params:

    NameTypeAttributeDescription
    contract String

    name of the contract
    classRef Class

    the class bind to this contract
    params Array
    • optional

    list of constructor parameters. Only if a parameter is a string and matches a contract, it will be replaced with the corresponding instance
    options Object
    • optional

    configuration
    options.noClassRef String
    • optional
    • default: false

    the reference is not a constructor but something else
    options.singleton String
    • optional
    • default: false

    create a new instance every time
    options.description String
    • optional

    describes the contract (currently only used by {{#crossLink "DI/listContracts:method"}}{{/crossLink}}).

    Return:

    Object

    this

    Example:

    App.di.registerType("ajax", App.AJAX) ;
App.di.registerType("ajax", App.AJAX, [], { singleton: true }) ;
App.di.registerType("util", App.Util, ["compress", true, ["wsql", "ls"] ], { singleton: true } ) ;
